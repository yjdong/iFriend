setQueryTimeout 方法 (ISQLServerStatement)

将驱动程序等待 SQLServerStatement 对象运行的秒数设置为给定的秒数。

语法

public final void setQueryTimeout(int seconds)

参数

seconds

指示要等待的秒数的 int 值，如果没有限制，则为 0。

异常

SQLServerException

备注

此 setQueryTimeout 方法由 java.sql.Statement 接口中的 setQueryTimeout 方法指定。
