getStatement 方法 (ISQLServerResultSet)

检索 SQLServerStatement 对象，该对象生成了此 SQLServerResultSet 对象。

语法

public java.sql.Statement getStatement()

返回值

SQLServerStatement 对象。

异常

SQLServerException

备注

此 getStatement 方法由 java.sql.ResultSet 接口中的 getStatement 方法指定。

如果通过其他方式生成结果集，例如使用 SQLServerDatabaseMetaData 方法，此方法将返回 Null。
