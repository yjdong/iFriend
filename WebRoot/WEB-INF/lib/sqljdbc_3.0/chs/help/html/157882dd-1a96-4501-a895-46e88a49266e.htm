setClob 方法 (int, java.io.Reader, long)

将指定参数设置为指定的具有给定字符数的 Reader 对象。

注意： 此功能是从 Microsoft SQL Server JDBC Driver 2.0 版开始引入的。

语法

public final void setClob(int parameterIndex,
java.io.Reader reader,
long length)

参数

parameterIndex

指示参数索引的 int。

reader

Reader 对象。

length

指示参数值中字符数的 long。

备注

此 setClob 方法由 java.sql.PreparedStatement 接口中的 setClob 方法指定。

异常

SQLServerException

请参阅

setClob 方法 (ISQLServerPreparedStatement)
