getSchemaName 方法 (SQLServerResultSetMetaData)

获取指定列的表架构名称。

语法

public java.lang.String getSchemaName(int column)

参数

column

指示列索引的 int 值。

返回值

包含架构名称的 String。

异常

SQLServerException

备注

此 getSchemaName 方法由 java.sql.ResultSetMetaData 接口中的 getSchemaName 方法指定。
