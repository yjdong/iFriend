getClientInfo 方法 (java.lang.String)

检索指定客户端信息属性的值。

注意： 从 Microsoft SQL Server JDBC Driver 2.0 版开始引入此功能。

语法

public java.lang.String getClientInfo (java.lang.String name)

参数

name

包含要检索的客户端信息属性名称的 String。

返回值

包含客户端信息属性值的 String。

异常

SQLServerException

备注

此 getClientInfo 方法由 java.sql.Connection 接口中的 getClientInfo 方法指定。

Microsoft SQL Server JDBC Driver 不支持任何客户端信息属性。 因此，此方法将返回 null。

与此类似，应用程序可以使用 SQLServerDatabaseMetaData 类中的 getClientInfoProperties 方法来检索驱动程序支持的客户端信息属性列表。 getClientInfoProperties 方法返回空结果集。
