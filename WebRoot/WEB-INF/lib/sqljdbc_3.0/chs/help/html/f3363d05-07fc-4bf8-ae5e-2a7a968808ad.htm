设置数据源属性

数据源是在 Java Platform, Enterprise Edition (Java EE) 环境中创建 JDBC 连接的首选机制。数据源可提供连接、池化连接和分布式连接，并且无需将连接属性硬编码到 Java 代码中。所有 Microsoft SQL Server JDBC Driver 数据源都可以分别使用相应的 setter 和 getter 方法，设置或获取任何属性的值。

通过使用 Java EE 产品（例如应用程序服务器和 servlet/JSP 引擎），通常可配置数据库访问的数据源。配置允许通过“属性=值”对来输入属性时，可以指定“设置连接属性”主题中列出的任何属性。

有关 SQL Server 数据源的详细信息，请参阅 SQLServerDataSource 类。有关如何使用 SQLServerDataSource 类来建立到 SQL Server 数据库的连接的实例，请参阅数据源示例。
