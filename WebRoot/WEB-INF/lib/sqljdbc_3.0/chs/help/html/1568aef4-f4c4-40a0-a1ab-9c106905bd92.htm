getClientInfoProperties 方法 (SQLServerDatabaseMetaData)

检索驱动程序支持的客户端信息属性列表。

注意： 此功能是从 Microsoft SQL Server JDBC Driver 2.0 版开始引入的。

语法

public java.sql.ResultSet getClientInfoProperties()

返回值

ResultSet 对象。

异常

SQLServerException

备注

此 getClientInfoProperties 方法由 java.sql.DatabaseMetaData 接口中的 getClientInfoProperties 方法指定。

注意： 此方法返回空结果集。驱动程序仅支持设置 applicationName，并且仅在连接时设置 applicationName。SQL Server 不支持在建立连接后更新客户端应用程序信息。
