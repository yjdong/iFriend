getFloat 方法 (int) (ISQLServerResultSet)

检索此 SQLServerResultSet 对象的当前行中指定列索引的值作为 Java 编程语言中的 float。

语法

public float getFloat(int columnIndex)

参数

columnIndex

指示列索引的 int 值。

返回值

float 值。

异常

SQLServerException

备注

此 getFloat 方法由 java.sql.ResultSet 接口中的 getFloat 方法指定。

此方法使用 Java float 精度返回基于数字的所有类型。
