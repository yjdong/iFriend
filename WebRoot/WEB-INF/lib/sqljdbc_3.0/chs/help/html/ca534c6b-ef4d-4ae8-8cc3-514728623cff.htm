executeUpdate 方法 ()

运行此 SQLServerPreparedStatement 对象中的 SQL 语句，这些语句必须是 SQL INSERT、UPDATE、MERGE 或 DELETE 语句；或是不返回任何内容的 SQL 语句，如 DDL 语句。

语法

public int executeUpdate()

返回值

一个指示受影响的行数的 int 值，如果使用 DDL 语句，则为 0。

异常

SQLServerException

备注

此 executeUpdate 方法由 java.sql.PreparedStatement 接口中的 executeUpdate 方法指定。
