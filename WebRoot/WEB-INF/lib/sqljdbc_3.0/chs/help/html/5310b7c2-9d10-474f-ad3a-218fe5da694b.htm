setFailoverPartner 方法 (SQLServerDataSource)

设置在数据库镜像配置中使用的故障转移服务器的名称。

语法

public void setFailoverPartner(java.lang.String serverName)

参数

serverName

包含故障转移服务器名称的 String 值。

备注

如果在与主服务器进行初始连接时失败，则使用由此方法设置的值；建立初始连接后，将忽略此值。此外，还应将此方法与 setDatabaseName 方法结合使用，否则将引发异常。

驱动程序不支持在设置故障转移服务器名称时指定故障转移服务器的端口号。但是，它支持在调用 setServerName 方法和 setInstanceName 方法时，同时调用 setFailoverPartner 方法。
