getMoreResults 方法 ()

移动到此 SQLServerStatement 对象的下一个结果。

语法

public final boolean getMoreResults()

返回值

如果返回的结果为一个结果集，则为 true。否则为 false。

异常

SQLServerException

备注

此 getMoreResults 方法由 java.sql.Statement 接口中的 getMoreResults 方法指定。

调用 getMoreResults 方法将隐式关闭使用 getResultSet 方法获取的当前打开的任何结果集对象。
