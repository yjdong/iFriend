getDouble 方法 (java.lang.String) (ISQLServerResultSet)

检索此 SQLServerResultSet 对象的当前行中指定列名称的值作为 Java 编程语言中的 double。

语法

public double getDouble(java.lang.String columnName)

参数

columnName

包含列名称的 String。

返回值

double 值。

异常

SQLServerException

备注

此 getDouble 方法由 java.sql.ResultSet 接口中的 getDouble 方法指定。

此方法使用 Java double 精度返回基于数字的所有数据类型。
