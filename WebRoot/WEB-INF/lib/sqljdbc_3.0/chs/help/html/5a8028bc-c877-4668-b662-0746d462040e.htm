getCharacterStream 方法 (long, long) (SQLServerNClob)

检索具有指定位置和长度且作为 Reader 对象或字符流的 NCLOB 数据。

注意： 从 Microsoft SQL Server JDBC Driver 2.0 版开始引入此功能。

语法

public java.io.Reader getCharacterStream(long pos,
long length)

参数

pos

指示与要检索的部分值的第一个字符之间的偏移量的 long。

length

指示要检索的部分值的字符长度的 long。

返回值

一个包含 NCLOB 数据的 Reader 对象。

异常

SQLServerException

备注

此 getCharacterStream 方法由 java.sql.NClob 接口中的 getCharacterStream 方法指定。
