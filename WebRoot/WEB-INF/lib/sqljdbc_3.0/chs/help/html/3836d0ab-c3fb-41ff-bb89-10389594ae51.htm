getSendStringParametersAsUnicode 方法 (SQLServerDataSource)

返回指示是否启用了以 UNICODE 格式将字符串参数发送到服务器的 boolean 值。

语法

public boolean getSendStringParametersAsUnicode()

返回值

如果以 UNICODE 格式将字符串参数发送到服务器，则为 true。否则为 false。

备注

如果将 sendStringParametersAsUnicode 属性设置为 true（默认值），则以 UNICODE 格式将字符串参数发送给服务器。如果将 sendStringParametersAsUnicode 设置为 false，则以 ASCII/MBCS 而非 UNICODE 格式将字符串参数发送给服务器。如果未设置 sendStringParametersAsUnicode 属性，则 getSendStringParametersAsUnicode 将返回默认值 true。

有关 sendStringParametersAsUnicode 连接属性的详细信息，请参阅设置连接属性。
