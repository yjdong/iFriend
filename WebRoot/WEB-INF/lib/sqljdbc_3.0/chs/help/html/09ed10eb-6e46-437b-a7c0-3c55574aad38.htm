isWrapperFor 方法 (SQLServerConnectionPoolDataSource)

指示此对象是否为指定接口的包装。

注意： 从 Microsoft SQL Server JDBC Driver 2.0 版开始引入此功能。

语法

public boolean isWrapperFor(Class iface)

参数

iface

定义接口的 class。

返回值

如果此对象实现了接口或包装了实现接口的对象，则为 true。否则为 false。

异常

SQLServerException

备注

isWrapperFor 方法和 unwrap 方法由在 JDBC 4.0 规范中引入的 java.sql.Wrapper 接口定义。

如果此方法返回 true，则将使用同一参数成功调用 unwrap。

有关详细信息，请参阅包装和接口。
