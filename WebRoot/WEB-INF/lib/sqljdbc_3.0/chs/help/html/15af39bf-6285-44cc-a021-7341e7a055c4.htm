getWarnings 方法 (ISQLServerConnection)

检索调用此 SQLServerConnection 对象时报告的第一个警告。

语法

public java.sql.SQLWarning getWarnings()

返回值

SQLWarning 对象。

异常

SQLServerException

备注

此 getWarnings 方法由 java.sql.Connection 接口中的 getWarnings 方法指定。

后续的警告链接到第一个 SQLWarning 并使用 getNextWarning 方法调用。如果在关闭的连接上调用，将引发异常。
