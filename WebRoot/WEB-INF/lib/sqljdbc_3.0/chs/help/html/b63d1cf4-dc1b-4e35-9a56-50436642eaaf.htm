setLoginTimeout 方法 (SQLServerDataSource)

设置此 SQLServerDataSource 对象在尝试连接时将等待的秒数。

语法

public void setLoginTimeout(int loginTimeout)

参数

loginTimeout

表示要等待的秒数的 int 值。零表示该超时为默认系统超时，默认情况下指定为 15 秒。

备注

此 setLoginTimeout 方法由 javax.sql.DataSource 接口中的 setLoginTimeout 方法指定。
