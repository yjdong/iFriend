setDateTimeOffset 方法 (SQLServerCallableStatement)

Microsoft SQL Server JDBC Driver 3.0 中新增了此方法。

将指定的列值设置为 DateTimeOffset 类 值。

语法

public void setDateTimeOffset(String sCol, microsoft.sql.DateTimeOffset t)

参数

sCol

列的名称。

t

DateTimeOffset 类 对象。

异常

SQLServerException

备注

可以使用 SQLServerCallableStatement.getDateTimeOffset 检索 DateTimeOffset 类 值。

setDateTimeOffset 获取列的序号。
