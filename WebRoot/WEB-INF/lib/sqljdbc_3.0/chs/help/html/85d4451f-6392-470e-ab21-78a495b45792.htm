isLast 方法 (ISQLServerResultSet)

检索游标是否位于此 SQLServerResultSet 对象的最后一行上。

语法

public boolean isLast()

返回值

如果游标在最后一行上，则为 true。如果游标在任何其他位置或结果集不包含任何行，则为 false。

异常

SQLServerException

备注

此 isLast 方法由 java.sql.ResultSet 接口中的 isLast 方法指定。

如果将此方法用于前进游标和动态游标，则将引发异常。
