updateDate 方法 (int, java.sql.Date)

根据给定的列索引使用日期值更新指定的列。

语法

public void updateDate(int index,
java.sql.Date x)

参数

index

指示列索引的 int 值。

x

日期值。

异常

SQLServerException

备注

此 updateDate 方法由 java.sql.ResultSet 接口中的 updateDate 方法指定。
