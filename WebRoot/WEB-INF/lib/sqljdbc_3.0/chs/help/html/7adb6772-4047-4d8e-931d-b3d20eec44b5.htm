rollback 方法 ()

撤消在当前事务中所做的所有更改并释放当前由此 SQLServerConnection 对象持有的任何数据库锁。

语法

public void rollback()

异常

SQLServerException

备注

此 rollBack 方法由 java.sql.Connection 接口中的 rollBack 方法指定。

仅当已禁用自动提交模式时才应使用此方法。
