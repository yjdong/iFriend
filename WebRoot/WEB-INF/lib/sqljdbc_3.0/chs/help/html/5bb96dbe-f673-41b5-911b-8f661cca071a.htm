prepareStatement 方法 (java.lang.String, int, int)

创建 SQLServerPreparedStatement 对象，该对象使用给定的类型和并发机制生成 SQLServerResultSet 对象。

语法

public java.sql.PreparedStatement prepareStatement(java.lang.String sSql,
int resultSetType,
int resultSetConcurrency)

参数

sSql

包含 SQL 语句的 String。

resultSetType

指示结果集类型的 int 值。

resultSetConcurrency

指示结果集并发类型的 int 值。

返回值

PreparedStatement 对象。

异常

SQLServerException

备注

此 prepareStatement 方法由 java.sql.Connection 接口中的 prepareStatement 方法指定。
