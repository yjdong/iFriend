setSavepoint 方法 ()

在当前事务中创建未命名的保存点，并返回代表它的新 SQLServerSavepoint 对象。

语法

public java.sql.Savepoint setSavepoint()

返回值

SavePoint 对象。

异常

SQLServerException

备注

此 setSavePoint 方法由 java.sql.Connection 接口中的 setSavePoint 方法指定。
