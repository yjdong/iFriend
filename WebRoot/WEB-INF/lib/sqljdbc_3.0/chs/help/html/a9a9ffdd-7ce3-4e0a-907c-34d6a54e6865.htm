getResponseBuffering 方法 (ISQLServerStatement)

检索此 SQLServerStatement 对象的响应缓冲模式。

语法

public final java.lang.String getResponseBuffering()

返回值

包含小写 full 或 adaptive 的 String。

备注

adaptive 指定在必要时缓冲尽可能少的数据。

full 指定在运行时从服务器读取全部结果。

adaptive 是 JDBC Driver 2.0 版和 3.0 版中的默认值。full 是 JDBC Driver 2.0 之前版本中的默认值。

有关使用响应缓冲模式的更多信息，请参阅使用自适应缓冲。
