executeUpdate 方法 (ISQLServerStatement)

运行给定的 SQL 语句，可以是 INSERT、UPDATE 或 DELETE 语句；或不返回任何内容的 SQL 语句，例如 SQL DDL 语句。从 Microsoft SQL Server JDBC Driver 3.0 开始，executeUpdate 将返回在 MERGE 操作中更新的正确行数。

重载列表

名称 说明

executeUpdate (java.lang.String)

运行给定的 SQL 语句，可以是 INSERT、UPDATE、DELETE 或 MERGE 语句；或不返回任何内容的 SQL 语句，例如 SQL DDL 语句。

executeUpdate (java.lang.String, int)

运行给定的 SQL 语句，并通过给定的标记向 Microsoft SQL Server JDBC 驱动程序发出信号，指示此 SQLServerStatement 对象自动生成的键是否应对检索可用。

executeUpdate (java.lang.String, int[])

运行给定的 SQL 语句，并向 JDBC 驱动程序发出信号，指示给定数组中指示的自动生成的键应对检索可用。

executeUpdate (java.lang.String, java.lang.String[])

运行给定的 SQL 语句，并向 JDBC 驱动程序发出信号，指示给定数组中指示的自动生成的键应对检索可用。
