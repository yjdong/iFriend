setTime 方法 (int, java.sql.Time, java.util.Calendar)

将指定参数设置为给定的时间和日历值。

语法

public final void setTime(int n,
java.sql.Time x,
java.util.Calendar cal)

参数

n

指示参数编号的 int 值。

x

Time 对象。

cal

Calendar 对象。

异常

SQLServerException

备注

此 setTime 方法由 java.sql.PreparedStatement 接口中的 setTime 方法指定。

从 SQL Server JDBC Driver 3.0 开始，此方法的行为由 sendTimeAsDatetime 连接属性（设置连接属性）和 SQLServerDataSource.setSendTimeAsDatetime 修改。

有关详细信息，请参阅配置如何将 java.sql.Time 值发送到服务器。
