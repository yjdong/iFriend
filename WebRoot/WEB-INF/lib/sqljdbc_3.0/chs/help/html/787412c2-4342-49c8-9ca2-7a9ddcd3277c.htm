position 方法 (byte[], long)

根据给定的 byte 数组模式和起始索引返回 BLOB 中指定模式的位置。

语法

public long position(byte[] bPattern,
long start)

参数

bPattern

要搜索的模式。

start

要搜索的起始位置。

返回值

指示找到模式的位置的 long 值，如果未找到，则为 -1。

异常

SQLServerException

备注

此 position 方法由 java.sql.Blob 接口中的 position 方法指定。
