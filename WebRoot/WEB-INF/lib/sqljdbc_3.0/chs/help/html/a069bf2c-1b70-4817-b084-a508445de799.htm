SSTRANSTIGHTLYCPLD

用于允许使用紧密结合的 XA 事务，这些事务具有不同的 XA 分支事务 ID (XID)，但具有相同的全局事务 ID (GTRID)。

方法

名称 说明

commit

提交由给定的 Xid 对象所指定的全局事务。

end

结束代表某个事务分支执行的工作。

forget

通知资源管理器忘记试探完成的事务分支。

getTransactionTimeout

获取为此 SQLServerXAResource 对象设置的当前事务超时值。

isSameRM

确定目标对象所表示的资源管理器实例是否与给定的 XAResource 对象所表示的资源管理器实例相同。

prepare

请求资源管理器为提交给定 Xid 对象所指定的事务做好准备。

recover

从资源管理器获取准备好的事务分支的列表。

rollback

请求资源管理器回滚代表事务分支执行的工作。

setTransactionTimeout

为此 SQLServerXAResource 对象设置当前事务超时值。

start

开始代表在 Xid 对象中指定的事务分支工作。
