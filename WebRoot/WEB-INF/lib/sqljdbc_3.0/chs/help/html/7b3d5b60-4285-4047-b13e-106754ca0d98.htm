executeUpdate 方法 (java.lang.String, int[])

运行给定的 SQL 语句，并向 Microsoft SQL Server JDBC 驱动程序发出信号，指示给定数组中指示的自动生成的键应对检索可用。

语法

public final int executeUpdate(java.lang.String sql,
int[] columnIndexes)

参数

sql

包含 SQL 语句的 String 。

columnIndexes

一个整数数组，指示应可用的自动生成的键的列索引。

返回值

一个指示受影响的行数的 int 值，如果使用 DDL 语句，则为 0。

异常

SQLServerException

备注

此 executeUpdate 方法由 java.sql.Statement 接口中的 executeUpdate 方法指定。
