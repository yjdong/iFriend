prepareCall 方法 (java.lang.String, int, int, int)

创建 SQLServerCallableStatement 对象，该对象使用给定的类型、并发机制和保持能力生成 SQLServerResultSet 对象。

语法

public java.sql.CallableStatement prepareCall(java.lang.String sql,
int nType,
int nConcur,
int nHold)

参数

sql

包含 SQL 语句的 String。

nType

指示结果集类型的 int 值。

nConcur

指示结果集并发类型的 int 值。

nHold

指示结果集保持能力的 int 值。

返回值

CallableStatement 对象。

异常

SQLServerException

备注

此 prepareCall 方法由 java.sql.Connection 接口中的 prepareCall 方法指定。
