length 方法 (SQLServerNClob)

检索 NClob 中的字符数。

注意： 从 Microsoft SQL Server JDBC Driver 2.0 版开始引入此功能。

语法

public long length()

返回值

NClob 的长度（以字符为单位）。

异常

SQLServerException

备注

此 length 方法由 java.sql.NClob 接口中的 length 方法指定。
