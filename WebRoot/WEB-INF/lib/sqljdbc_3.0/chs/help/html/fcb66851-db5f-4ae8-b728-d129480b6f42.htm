getURL 方法 (SQLServerDatabaseMetaData)

检索此数据库的 URL。

语法

public java.lang.String getURL()

返回值

包含 URL 的 String。

异常

SQLServerException

备注

此 getURL 方法由 java.sql.DatabaseMetaData 接口中的 getURL 方法指定。

将 Microsoft SQL Server JDBC Driver 与 SQL Server 数据库一起使用时，此方法会返回包含以下信息的 String 值：

  • "jdbc:sqlserver://" 的 URL 值
  • 可选连接属性，如 serverName、instanceName 和 portNumber
  • 由用户设置的其他连接属性和设为非空或非 Null 的驱动程序默认值的所有连接属性，不包括 userName、password 和 integratedSecurity。
