deleteRow 方法 (ISQLServerResultSet)

从此 SQLServerResultSet 对象和基础数据库中删除当前行。

语法

public void deleteRow()

异常

SQLServerException

备注

此 deleteRow 方法由 java.sql.ResultSet 接口中的 deleteRow 方法指定。

游标位于插入行时，无法调用此方法。

使用键集游标时，此方法在结果集中留有一个孔。可以通过使用 rowDeleted 方法测试此孔。结果集中的行的行号不变。
