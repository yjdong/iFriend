setSQLXML 方法 (SQLServerCallableStatement)

将指定参数设置为指定的 SQLXML 对象。

注意： 从 Microsoft SQL Server JDBC Driver 2.0 版开始引入此功能。

语法

public final void setSQLXML(java.lang.String parameterName,
java.sql.SQLXML xmlObject)

参数

parameterName

指示参数名称的 String。

xmlObject

包含参数值的 SQLXML 对象。

异常

SQLServerException

备注

此 setSQLXML 方法由 java.sql.CallableStatement 接口中的 setSQLXML 方法指定。
