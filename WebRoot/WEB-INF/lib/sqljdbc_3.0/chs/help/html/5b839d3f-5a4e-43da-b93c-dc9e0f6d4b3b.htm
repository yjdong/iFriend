getArray 方法 (int)

根据给定的参数索引检索指定参数的值作为 Array 对象。

语法

public java.sql.Array getArray(int i)

参数

i

指示参数索引的 int 值。

返回值

Array 对象。

异常

SQLServerException

备注

此 getArray 方法由 java.sql.CallableStatement 接口中的 getArray 方法指定。
