relative 方法 (ISQLServerResultSet)

将游标沿正向或反向移动相对于当前行的指定行数。

语法

public boolean relative(int nRows)

参数

nRows

指示要移动的行数的 int 值。

返回值

如果游标在行上，则为 true。否则为 false。

异常

SQLServerException

备注

此 relative 方法由 java.sql.ResultSet 接口中的 relative 方法指定。

如果尝试移到结果集中的第一行之前或最后一行之后，将使游标移到第一行之前或最后一行之后。调用 relative(0) 有效，但是不会更改游标位置。

调用 relative(1) 方法与调用 next 方法效果相同。调用 relative(-1) 方法与调用 previous 方法效果相同。
