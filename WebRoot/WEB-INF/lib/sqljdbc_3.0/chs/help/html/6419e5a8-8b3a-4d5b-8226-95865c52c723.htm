getMoreResults 方法 (int)

移到此 SQLServerStatement 对象的下一个结果并根据给定模式所指定的说明处理任何当前打开的结果集对象。

语法

public final boolean getMoreResults(int mode)

参数

mode

指示如何处理当前打开的结果集对象的 int 值。它必须是下列常量之一：

CLOSE_CURRENT_RESULT

KEEP_CURRENT_RESULT（JDBC 驱动程序不支持）

CLOSE_ALL_RESULTS

返回值

如果返回的结果为一个结果集，则为 true。否则为 false。

异常

SQLServerException

备注

此 getMoreResults 方法由 java.sql.Statement 接口中的 getMoreResults 方法指定。

如果在检索结果前调用 getMoreResults 方法，则该方法就会采用 mode 参数指定的行为方式并移到下一个结果。

注意： JDBC 驱动程序不支持使用 KEEP_CURRENT_RESULT 常量。如果使用该常量，将引发异常。
