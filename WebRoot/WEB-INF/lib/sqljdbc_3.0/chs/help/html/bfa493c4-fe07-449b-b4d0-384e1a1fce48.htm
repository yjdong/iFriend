getString 方法 (int) (ISQLServerResultSet)

检索此 SQLServerResultSet 对象的当前行中指定列索引的值作为 Java 编程语言中的 String。

语法

public java.lang.String getString(int columnIndex)

参数

columnIndex

指示列索引的 int 值。

返回值

String 值。

异常

SQLServerException

备注

此 getString 方法由 java.sql.ResultSet 接口中的 getString 方法指定。

SQL Server 中的所有列都可作为字符串返回。这意味着可以返回基于数字和基于字符的所有类型的 String 表示形式，以及二进制列（如 binary、varbinary、varbinary(max)、image、timestamp 和 uniqueidentifier）的十六进制字符串表示形式。

位置敏感类型（如 money、smallmoney, datetime、smalldatetime、float、real、decimal 和 numeric）将返回基本类型值的规范 toString() 格式。

用户定义类型将作为十六进制 String 值返回。
