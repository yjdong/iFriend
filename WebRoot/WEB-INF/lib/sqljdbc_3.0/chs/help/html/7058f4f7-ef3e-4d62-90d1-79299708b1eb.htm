SQLServerClob 构造函数 (SQLServerConnection, java.lang.String)

在给定 SQLServerConnection 对象和数据字符串时，初始化 SQLServerClob 类的新实例。

注意： JDBC 驱动程序 2.0 版已不推荐使用此方法，而是使用 SQLServerConnection 类的 createClob 方法。

语法

public SQLServerClob(SQLServerConnection connection,
java.lang.String data)

参数

connection

SQLServerConnection 对象。

data

CLOB 数据。
