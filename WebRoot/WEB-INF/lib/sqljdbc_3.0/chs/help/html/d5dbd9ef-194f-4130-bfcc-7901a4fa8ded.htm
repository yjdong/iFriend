rollback 方法 (java.sql.Savepoint)

撤消在设置给定的 SQLServerSavepoint 对象后所做的所有更改。

语法

public void rollback(java.sql.Savepoint s)

参数

s

要回滚到的 SavePoint 对象。

异常

SQLServerException

备注

此 rollBack 方法由 java.sql.Connection 接口中的 rollBack 方法指定。

仅当已禁用自动提交模式时才应使用此方法。
