getConnection 方法 (java.lang.String, java.lang.String)

使用给定的用户名和密码尝试与此 SQLServerDataSource 对象表示的数据源建立连接。

语法

public java.sql.Connection getConnection(java.lang.String username,
java.lang.String password)

参数

username

包含用户名的 String。

password

包含密码的 String。

返回值

SQLServerConnection 对象。

异常

java.sql.SQLException

备注

此 getConnection 方法由 javax.sql.DataSource 接口中的 getConnection 方法指定。

如果使用非 Null 的用户名或密码调用 getConnection 方法，将替换初始化 SQLServerConnection 对象时为 SQLServerDataSource 类设置的用户名和密码属性。例如，如果调用方对数据源调用了 setUser 和 setPassword，然后调用 getConnection 并提供非 Null 的用户名或密码，由 setUser 和 setPassword 设置的用户名和密码将被替换为传入 getConnection 的用户名和密码。

注意： 在这种情况下，通过调用 setURL 方法在 URL 中设置的用户名和密码将保持不变。
