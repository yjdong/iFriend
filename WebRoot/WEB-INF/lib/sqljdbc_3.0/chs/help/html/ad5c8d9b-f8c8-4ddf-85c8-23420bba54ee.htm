updateNClob 方法 (java.lang.String, java.io.Reader, long)

使用指定的具有指定字符数的 Reader 对象更新指定列。

注意： 此功能是从 Microsoft SQL Server JDBC Driver 2.0 版开始引入的。

语法

public void updateNClob(java.lang.String columnLabel,
java.io.Reader reader,
long length)

参数

columnLabel

指示列标签的 String。

reader

Reader 对象。

length

参数数据中的字符数。

异常

SQLServerException

备注

此 updateNClob 方法由 java.sql.ResultSet 接口中的 updateNClob 方法指定。

只在 nvarchar(max)、ntext 和 xml 列上支持此方法。在任何其他数据类型上使用此方法会引发异常。
