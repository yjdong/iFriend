getSQLXML 方法 (int)

在给定参数索引时检索指定参数的值作为 SQLXML 对象。

注意： 此功能是从 Microsoft SQL Server JDBC Driver 2.0 版开始引入的。

语法

public final java.sql.SQLXML getSQLXML(int parameterIndex)

参数

parameterIndex

指示参数索引的 int。

返回值

SQLXML 对象。

异常

SQLServerException

备注

此 getSQLXML 方法由 java.sql.CallableStatement 接口中的 getSQLXML 方法指定。
