setPoolable 方法 (ISQLServerStatement)

请求语句入池或不入池。

语法

public void setPoolable(boolean poolable) throws SQLException

参数

poolable

如果为 true，则请求语句入池。如果为 false，则请求语句不入池。

异常

SQLServerException

备注

poolable 参数中指定的值是对语句池实现的提示，用于指示应用程序是否希望语句入池。语句池管理器决定应用程序是否使用该提示。

语句的池值既应用于由驱动程序实现的内部语句缓存，也应用于由应用程序服务器和其他应用程序实现的外部语句缓存。

默认情况下，SQLServerStatement 对象在创建后不可入池。SQLServerPreparedStatement 和 SQLServerCallableStatement 对象在创建后可入池。

如果对已关闭的语句调用此方法，将引发 SQLServerException。

isPoolable 返回指示对象是否可入池的值。
