setReadOnly 方法 (ISQLServerConnection)

将此 SQLServerConnection 对象设置为只读模式，以此作为 JDBC 驱动程序启用数据库优化的提示。

注意： Microsoft SQL Server JDBC Driver 不支持此方法。

语法

public void setReadOnly(boolean readOnly)

参数

readOnly

如果连接为只读，则为 true。否则为 false。

异常

SQLServerException

备注

此 setReadOnly 方法由 java.sql.Connection 接口中的 setReadOnly 方法指定。
