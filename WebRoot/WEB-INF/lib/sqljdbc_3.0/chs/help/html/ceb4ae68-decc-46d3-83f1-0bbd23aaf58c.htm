getFetchDirection 方法 (ISQLServerStatement)

检索从数据库表中提取行的方向，该方向是由此 SQLServerStatement 对象生成的结果集的默认值。

注意： Microsoft SQL Server JDBC Driver 目前不能实现此方法。因此，它将始终返回 FETCH_UNKNOWN。

语法

public final int getFetchDirection()

返回值

指示提取方向的 int 值，该提取方向由 setFetchDirection 方法指定。

异常

SQLServerException

备注

此 getFetchDirection 方法由 java.sql.Statement 接口中的 getFetchDirection 方法指定。
