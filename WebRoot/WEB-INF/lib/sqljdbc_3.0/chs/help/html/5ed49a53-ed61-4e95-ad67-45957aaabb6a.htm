getMetaData 方法 (ISQLServerPreparedStatement)

检索一个 SQLServerResultSetMetaData 类 对象，该对象包含有关运行此 SQLServerPreparedStatement 对象时将返回的 SQLServerResultSet 对象的列信息。

语法

public final java.sql.ResultSetMetaData getMetaData()

返回值

ResultSetMetaData 对象。

异常

SQLServerException

备注

此 getMetaData 方法由 java.sql.PreparedStatement 接口中的 getMetaData 方法指定。
