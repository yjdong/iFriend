setXopenStates 方法 (SQLServerDataSource)

设置指示是否启用了将 SQL 状态转换为 XOPEN 兼容状态的 boolean 值。

语法

public void setXopenStates(boolean xopenStates)

参数

xopenStates

如果启用了将 SQL 状态转换为 XOPEN 兼容状态，则为 true。否则为 false。

备注

如果将 xopenStates 属性设置为 true，则 Microsoft SQL Server JDBC 驱动程序会将 SQL 状态转换为 XOPEN 兼容状态。默认为 false，这将导致 JDBC 驱动程序生成 SQL 99 状态代码。如果未设置 xopenStates，则 getXopenStates 方法会返回默认值 false。
