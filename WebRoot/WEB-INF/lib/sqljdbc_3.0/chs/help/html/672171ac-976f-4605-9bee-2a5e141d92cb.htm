getSchemas 方法 (String, String)

通过使用指定目录名称和架构名称来检索当前数据库中可用的架构名称。

注意： 从 Microsoft SQL Server JDBC Driver 2.0 版开始引入此功能。

语法

public ResultSet getSchemas(java.lang.String catalog,
java.lang.String schemaPattern)

参数

catalog

数据库中的目录名称。如果该名称为空字符串 ""，则结果将包括无目录的架构。如果该名称为 null，则目录名称不用于搜索。

schemaPattern

架构的名称。如果该名称为 null，则架构名称不用于搜索。

返回值

SQLServerResultSet 对象。

异常

SQLServerException

备注

此 getSchemas 方法由 java.sql.DatabaseMetaData 接口中的 getSchemas 方法指定。

由 getSchemas 方法返回的结果集会包含下列信息：

名称 类型 说明

TABLE_SCHEM

String

架构的名称。

TABLE_CATALOG

String

架构的目录名称。

结果先按 TABLE_CATALOG 再按 TABLE_SCHEM 排序。每个行都以 TABLE_SCHEM 作为第一列并以 TABLE_CATALOG 作为第二列。
