获取驱动程序版本

可以通过以下方法找到所安装的 Microsoft SQL Server JDBC Driver 的版本：

  • 调用 SQLServerDatabaseMetaData 方法 getDriverMajorVersion、getDriverMinorVersion 或 getDriverVersion。
  • 版本将显示在产品分发内容的 readme.txt 文件中。

此外，可以通过在 SQLServerDatabaseMetaData 类上调用 getDriverName 方法来返回 JDBC Driver 名称。将返回“Microsoft SQL Server JDBC Driver 3.0”。

以下是调用 SQLServerDatabaseMetaData 类的各个方法所输出的内容示例：

getDriverName = Microsoft SQL Server JDBC Driver 3.0

getDriverMajorVersion = 3

getDriverMinorVersion = 0

getDriverVersion = 3.0. xxx.x

其中，“xxx.x”是最终版本号。
