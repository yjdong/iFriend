execute 方法 (java.lang.String, int)

运行可返回多项结果的给定的 SQL 语句，并通知 Microsoft SQL Server JDBC Driver 以指明任何自动生成的键应可用于检索。

语法

public final boolean execute(java.lang.String sql,
int flag)

参数

sql

包含 SQL 语句的 String。

flag

指示是否应使用自动生成的键的 int 值。它必须是下列常量之一：

RETURN_GENERATED_KEYS

NO_GENERATED_KEYS

返回值

如果第一个结果为一个结果集，则为 true。否则为 false。

异常

SQLServerException

备注

此 execute 方法由 java.sql.Statement 接口中的 execute 方法指定。
