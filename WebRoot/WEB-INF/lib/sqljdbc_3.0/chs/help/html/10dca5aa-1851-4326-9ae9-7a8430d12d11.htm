setLockTimeout 方法 (SQLServerDataSource)

设置指示数据库在报告锁定超时之前要等待的毫秒数的 int 值。

语法

public void setLockTimeout(int lockTimeout)

参数

lockTimeout

包含要等待的毫秒数的 int 值。

备注

锁定超时值是数据库报告锁定超时之前要等待的毫秒数。默认值 -1 表示可以无限期等待。如果指定，该值将成为此连接上所有语句的默认值。

注意： 值为 0 表示不等待。如果未设置 lockTimeout 属性，getLockTimeout 方法将返回默认值 -1。
