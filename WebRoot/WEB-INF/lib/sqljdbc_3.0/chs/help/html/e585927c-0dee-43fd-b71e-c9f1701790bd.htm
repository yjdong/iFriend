getDateTimeOffset(java.lang.string) (SQLServerResultSet)

Microsoft SQL Server JDBC Driver 3.0 中新增了此方法。

根据给定的参数索引，检索指定列的值作为 Java 编程语言中的 DateTimeOffset 类 对象。

语法

public microsoft.sql.DateTimeOffset getDateTimeOffset(String columnName)

参数

columnName

列的名称。

返回值

DateTimeOffset 类 对象。

异常

SQLServerException

备注

可以使用 SQLServerResultSet.updateDateTimeOffset 更新 DateTimeOffset 类 值。
