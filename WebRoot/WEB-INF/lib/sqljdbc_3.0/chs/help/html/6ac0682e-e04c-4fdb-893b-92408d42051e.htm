setEscapeProcessing 方法 (ISQLServerStatement)

设置转义处理模式。

注意： Microsoft SQL Server JDBC 驱动程序的转义处理功能始终启用。将此方法设置为 false 将不起作用。

语法

public final void setEscapeProcessing(boolean enable)

参数

enable

如果为 true，则启用转义处理。否则为 false。

异常

SQLServerException

备注

此 setEscapeProcessing 方法由 java.sql.Statement 接口中的 setEscapeProcessing 方法指定。
