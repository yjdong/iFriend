setFetchDirection 方法 (ISQLServerStatement)

为 Microsoft SQL Server JDBC 驱动程序提供提示以指明处理结果集行时应采用的方向。

注意： JDBC 驱动程序目前忽略了此方法给出的提示。

语法

public final void setFetchDirection(int nDir)

参数

nDir

指示行处理方向的 int 值，可以为以下值之一：

FETCH_FORWARD

FETCH_REVERSE

FETCH_UNKNOWN

异常

SQLServerException

备注

此 setFetchDirection 方法由 java.sql.Statement 接口中的 setFetchDirection 方法指定。
