SQLServerConnection 类

表示到 Microsoft SQL Server 数据库的 JDBC 连接。

包：com.microsoft.sqlserver.jdbc

实现：ISQLServerConnection、java.io.Serializable

语法

public class SQLServerConnection

备注

SQLServerConnection 支持 JDBC 连接池并且可为物理 JDBC 连接或逻辑 JDBC 连接。SQLServerConnection 管理自其创建的所有语句的事务控制，而且可参与通过 XAResource 适配器管理的 XA 分布式事务。

SQLServerConnection 管理预定义语句句柄池。预定义语句仅准备一次，但通常使用不同的语句参数数据值运行多次。预定义语句还可跨多个关闭的逻辑（入池）连接进行维护。

注意： SQLServerConnection 不是线程安全的。但是，可以在并发线程中同时处理基于单个连接创建的多个语句。

此类支持取消对 SQLServerConnection 类、 java.sql.connection 接口和 ISQLServerConnection 接口的包装。有关详细信息，请参阅包装和接口。
