getParameterClassName 方法 (SQLServerParameterMetaData)

检索 Java 类的完全限定名称，而该 Java 类的实例应传递给 SQLServerPreparedStatement 类中的 setObject 方法。

语法

public java.lang.String getParameterClassName(int param)

参数

param

指示参数索引的 int 值。

返回值

包含完全限定的类名称的 String。

异常

SQLServerException

备注

此 getParameterClassName 方法由 java.sql.ParameterMetaData 接口中的 getParameterClassName 方法指定。
