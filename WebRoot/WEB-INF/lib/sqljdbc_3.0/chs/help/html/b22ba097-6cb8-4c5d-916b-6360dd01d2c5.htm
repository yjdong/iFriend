getByte 方法 (int) (ISQLServerResultSet)

检索此 SQLServerResultSet 对象的当前行中指定列索引的值作为 Java 编程语言中的 byte。

语法

public byte getByte(int columnIndex)

参数

columnIndex

指示列索引的 int 值。

返回值

byte 值。

异常

SQLServerException

备注

此 getByte 方法由 java.sql.ResultSet 接口中的 getByte 方法指定。

只有可以安全返回 byte 值的 SQL Server 数据类型（例如 tinyint 和 bit）才支持此方法。任何其他数据类型将会引发异常。
