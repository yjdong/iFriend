isPoolable 方法 (ISQLServerStatement)

返回指示是否可以将语句添加到用户提供的语句池的一个值。

语法

public boolean isPoolable() throws SQLException

返回值

如果可以将该语句添加到用户提供的语句池，则为 true；否则为 false。

异常

SQLServerException

备注

setPoolable 更改对象的可入池行为。
