setCursorName 方法 (ISQLServerStatement)

将 SQL 游标名称设置为给定字符串，该字符串将由随后的执行方法使用。

注意： Microsoft SQL Server JDBC Driver 目前不支持此方法。调用此方法将不起作用。

语法

public final void setCursorName(java.lang.String name)

参数

name

一个包含游标名称的 String 值。

异常

SQLServerException

备注

此 setCursorName 方法由 java.sql.Statement 接口中的 setCursorName 方法指定。
