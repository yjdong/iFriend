setClientInfo 方法 (java.lang.String, java.lang.String)

设置指定客户端信息属性的值。

注意： 此功能是从 Microsoft SQL Server JDBC Driver 2.0 版开始引入的。

语法

public void setClientInfo (java.lang.String name,
java.lang.String value)

参数

name

包含要设置的客户端信息属性名称的 String。

value

包含客户端信息属性设置值的 String。

异常

SQLServerException

备注

此 setClientInfo 方法由 java.sql.Connection 接口中的 setClientInfo 方法指定。

Microsoft SQL Server JDBC Driver 不支持任何客户端信息属性。在 JDBC Driver 2.0 发行版中，此方法生成属性的警告。应用程序应使用 SQLServerConnection 类的 getWarnings 方法来检索警告。
