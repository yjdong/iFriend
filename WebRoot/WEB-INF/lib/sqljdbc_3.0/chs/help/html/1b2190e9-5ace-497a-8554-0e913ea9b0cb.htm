setString 方法 (long, java.lang.String)

将给定的 String 写入 CLOB（从指定位置开始）。

语法

public int setString(long pos,
java.lang.String s)

参数

pos

要开始写入 CLOB 的位置。

s

要写入到 CLOB 的 String。

返回值

写入的字符数。

异常

java.sql.SQLException

备注

setString 方法由 java.sql.Clob 接口中的 setString 方法指定。

从指定位置开始覆盖字符数据，并可以超过 CLOB 的初始长度。指定“位置+1”值将追加到字符串末尾。指定“位置+2”或更大值（或零或更小值）会引发位置错误。
