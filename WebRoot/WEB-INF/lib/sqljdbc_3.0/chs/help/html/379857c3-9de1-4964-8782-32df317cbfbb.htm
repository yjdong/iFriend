SSTRANSTIGHTLYCPLD 字段 (SQLServerXAResource)

用于允许使用紧密结合的 XA 事务，这些事务具有不同的 XA 分支事务 ID (XID)，但具有相同的全局事务 ID (GTRID)。

语法

public static final int SSTRANSTIGHTLYCPLD

字段值

为 32768 的 int 值。

备注

每个事务都由一个 XA 分支事务 ID (XID) 和全局事务 ID (GTRID) 标识。为了允许应用程序使用紧密结合的 XA 事务，这些事务具有不同的 XID 但具有相同的 GTRID，您必须为 XAResource.start 方法的 flags 参数设置 SSTRANSTIGHTLYCPLD。有关如何使用此标志的详细信息，请参阅了解 XA 事务。
