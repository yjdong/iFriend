getBinaryStream 方法 (long, long)

通过使用指定的起始位置和长度来返回包含部分 BLOB 值的输入流对象。

注意： 从 Microsoft SQL Server JDBC Driver 2.0 版开始引入此功能。

语法

public java.io.InputStream getBinaryStream(long pos, long length)

参数

pos

要检索的部分值的第一个字节的偏移量。

length

要检索的部分值的字节长度。

返回值

包含 BLOB 数据的输入流。

异常

SQLServerException

备注

此 getBinaryStream 方法由 java.sql.Blob 接口中的 getBinaryStream 方法指定。
