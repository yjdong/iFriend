getNClob 方法 (int)

检索指定 JDBC NCLOB 参数的值作为 Java 编程语言中的 NClob 对象。

注意： 此功能是从 Microsoft SQL Server JDBC Driver 2.0 版开始引入的。

语法

public java.sql.NClob getNClob(int parameterIndex)

参数

parameterIndex

指示参数索引的 int。

返回值

NClob 对象。

异常

SQLServerException

备注

此 getNClob 方法由 java.sql.CallableStatement 接口中的 getNClob 方法指定。

此方法仅支持检索 NCHAR、NVARCHAR、NTEXT 和 XML 参数。在其他数据类型参数上调用这些方法会引发异常。
