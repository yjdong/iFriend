jdbcCompliant 方法 (SQLServerDriver)

验证 Microsoft SQL Server JDBC 驱动程序是否符合 JDBC 规范。

语法

public boolean jdbcCompliant()

返回值

如果 JDBC 驱动该程序满足最低要求，则为 true。否则为 false。

备注

此 jdbcCompliant 方法由 java.sql.Driver 接口中的 jdbcCompliant 方法指定。
