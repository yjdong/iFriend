getDateTimeOffset 方法 (int)

Microsoft SQL Server JDBC Driver 3.0 中新增了此方法。

根据给定的参数索引，检索指定参数的值作为 Java 编程语言中的 DateTimeOffset 类 对象。

语法

public microsoft.sql.DateTimeOffset getDateTimeOffset(int index)

参数

index

从 1 开始的参数序号。

返回值

DateTimeOffset 类 对象。

异常

SQLServerException

备注

可以使用 SQLServerCallableStatement.setDateTimeOffset 设置 DateTimeOffset 类 参数值。
