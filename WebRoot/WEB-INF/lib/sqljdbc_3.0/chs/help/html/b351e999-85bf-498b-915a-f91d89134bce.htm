getNString 方法 (java.lang.String)

检索指定 NCHAR、NVARCHAR 或 LONGNVARCHAR 参数的值作为 Java 编程语言中的 String。

注意： 此功能是从 Microsoft SQL Server JDBC Driver 2.0 版开始引入的。

语法

public final java.lang.String getNString(java.lang.String parameterName)

参数

parameterName

包含参数名称的 String。

返回值

String 对象。

异常

SQLServerException

备注

此 getNString 方法由 java.sql.CallableStatement 接口中的 getNString 方法指定。
