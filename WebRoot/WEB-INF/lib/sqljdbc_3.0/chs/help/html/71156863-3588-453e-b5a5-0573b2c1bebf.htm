isWrapperFor 方法 (SQLServerCallableStatement)

指示此语句对象是否为指定接口的包装。

注意： 此功能是从 Microsoft SQL Server JDBC Driver 2.0 版开始引入的。

语法

public boolean isWrapperFor(Class iface)

参数

iface

定义接口的 class。

返回值

如果此对象实现了接口或包装了实现接口的对象，则为 true。否则为 false。

异常

SQLServerException

备注

isWrapperFor 方法和 unwrap 方法由在 JDBC 4.0 中引入的 java.sql.Wrapper 接口定义。

如果此方法返回 true，则能够使用同一参数成功调用 unwrap。

有关详细信息，请参阅包装和接口。
