getNString 方法 (java.lang.String) (ISQLServerResultSet)

检索 SQLServerResultSet 对象的当前行中指定列的值作为 java.lang.String 对象。

注意： 此功能是从 Microsoft SQL Server JDBC Driver 2.0 版开始引入的。

语法

public java.lang.String getNString(java.lang.String columnLabel)

参数

columnLabel

包含列标签的 String。

返回值

String 对象。

异常

SQLServerException

备注

此 getNString 方法由 java.sql.SQLServerResultSet 接口中的 getNString 方法指定。

此方法可用于检索此 SQLServerResultSet 对象的当前行的 nvarchar、nchar、nvarchar(max)、ntext 或 xml 列的值。如果尝试使用此方法检索其他数据类型的值，则会引发异常。
