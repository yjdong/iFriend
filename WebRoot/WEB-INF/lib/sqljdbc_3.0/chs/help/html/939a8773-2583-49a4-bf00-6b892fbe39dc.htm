JDBC 驱动程序概述

Microsoft SQL Server JDBC 驱动程序是与 Java 数据库连接 (JDBC) 4.0 兼容的驱动程序，可提供对 Microsoft SQL Server 数据库的可靠数据访问。JDBC 驱动程序可访问 SQL Server 2005 中引入的许多功能，包括数据库镜像、xml 数据类型、用户定义的数据类型以及大值数据类型；并且支持新的“快照”事务隔离。此外，JDBC 驱动程序还支持使用集成身份验证。在 SQL Server JDBC Driver 3.0 中，新增了对一些 SQL Server 2008 功能的支持。有关详细信息，请参阅 JDBC 驱动程序的新增功能。

此部分中的主题提供了 JDBC 驱动程序的全面概述，包括使用它所需满足的系统要求、使用方法以及在何处可获取更多信息。
