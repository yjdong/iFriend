close 对象 (ISQLServerResultSet)

立即释放此 SQLServerResultSet 对象的数据库和 JDBC 资源，而不是在对象自动关闭时，等待上述操作发生。

语法

public void close()

异常

SQLServerException

备注

此 close 方法由 java.sql.ResultSet 接口中的 close 方法指定。

SQLServerStatement 对象会自动关闭其在 SQLServerStatement 对象关闭、重新运行或用于从一个多结果序列中检索下一个结果时生成的 SQLServerResultSet 对象。对 SQLServerResultSet 对象进行垃圾收集处理后，此对象会自动关闭。

当执行可生成单个大型只进、只读结果集的语句时，您所感兴趣的可能只是返回的结果集中的一些初始行集。在此情况下，应用程序可能会在关闭结果集之前调用关联语句对象的 cancel 方法，以最大程度地减少放弃余下不必要的行所需要的处理时间。在决定是否应使用此方法时，建议对照可节约的处理时间与取消执行所需的时间及与服务器之间的附加往返，在两者之间进行折衷。
