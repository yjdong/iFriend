getSendTimeAsDatetime 方法 (SQLServerDataSource)

SQL Server JDBC Driver 3.0 中新增了此方法。

返回 sendTimeAsDatetime 连接属性的设置。

语法

public boolean getSendTimeAsDatetime();

返回值

如果将 java.sql.Time 值作为 SQL Server datetime 类型发送到服务器，则为 true。如果将 java.sql.Time 值作为 SQL Server time 类型发送到服务器，则为 false。

备注

有关 sendTimeAsDatetime 连接属性的详细信息，请参阅设置连接属性。

SQLServerDataSource.setSendTimeAsDatetime 允许您通过编程设置 sendTimeAsDatetime 连接属性。

有关详细信息，请参阅配置如何将 java.sql.Time 值发送到服务器。
